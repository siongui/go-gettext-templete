===================
Go Template Utility
===================

Utility for Go_ tempalte (`text/template`_ and `html/template`_).

Development Environment:

  - `Ubuntu 16.10`_
  - `Go 1.7.5`_


Install
+++++++

.. code-block:: bash

  go get -u github.com/siongui/gotemplateutil
